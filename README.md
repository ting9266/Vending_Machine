# Vending_Machine

## 開發平台:
Windows 11

## 開發環境:
ModelSim-Altera 10.1d (Quartus II 13.1)



## 題目說明:
利用有限狀態機(FSM)的方法模擬出自動販賣機。分為4個狀態:開始投錢(初始狀態，S0)、當投入金額達飲料最低價格時，開始顯示可購買飲料名稱(選擇狀態S1)、當選擇飲料後輸出飲料名(給予狀態，S2)、找零(結帳狀態，S3)，過程中可隨時取消購買，退出所有零錢並回到初始狀態(S0)。
